# Vaibhav P - Portfolio Website

This project is a modern personal portfolio website developed using HTML, CSS, and JavaScript. The portfolio is designed to present personal information, technical skills, education details, and contact information in a clean and interactive way.

The website follows a futuristic luxury themed design with smooth animations, glowing UI elements, responsive layouts, and interactive sections. The interface is built to provide a modern browsing experience while maintaining simplicity and readability.

The homepage introduces the profile with a visually engaging hero section along with animated dashboard elements and smooth transition effects. The portfolio also contains an about section, education timeline, skills section, and a contact section for communication.

The project is fully responsive and adapts across desktop and mobile devices. CSS is used extensively for styling, gradients, animations, and glassmorphism effects, while JavaScript is used to add interactivity, loading animations, scroll effects, and dynamic UI behavior.

This portfolio project was created as a frontend web development project to showcase modern web design concepts and user interface implementation using core web technologies.

## Technologies Used
- HTML5
- CSS3
- JavaScript
